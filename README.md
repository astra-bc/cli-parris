## インストール

### 必要なもの

- Python 3.8 以上
- macOS（サウンド再生に `afplay` を使用）
- ターミナルの幅 80 文字以上推奨

### GitHub からインストール

```bash
pip install git+https://github.com/astra-bc/cli-parris.git
```

### ソースからインストール

```bash
git clone https://github.com/astra-bc/cli-parris.git
cd cli-parris
pip install .
```

### 開発用インストール

```bash
git clone https://github.com/astra-bc/cli-parris.git
cd cli-parris
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
```

## 遊び方

### 起動

```bash
nononono2
```

または:

```bash
python -m cli_parris
```

> **注意**: 起動前に **半角英数入力** に切り替えてください。

### タイトル画面

`J` キーでゲーム開始。`Q` で終了。

## 操作方法

ホームポジションで全操作が完結します。

```
 左手                右手
┌─────────┐    ┌──────────────┐
│ A: アイテム←  │    │ J: パリィ (メイン)  │
│ S: アイテム使用│    │ ;: 残火の業火      │
│ D: アイテム→  │    │ SPACE: 攻撃        │
│ F: パリィ (サブ)│    │                    │
└─────────┘    └──────────────┘
```

| キー | アクション |
|------|-----------|
| `J` | パリィ：メインレーン（メロディー） |
| `F` | パリィ：サブレーン（8ビートリズム） |
| `Space` | 攻撃 / 残火MAX時は長押しで残火バースト |
| `;` | 残火の業火（残火MAX＋敵HP1で●を削る） |
| `A` / `D` | アイテム選択（左右） |
| `S` | アイテム使用 |
| `R` | リスタート（ゲームオーバー/クリア時） |
| `Q` | 終了 |

## ゲームシステム

### 2レーン・パリィ

画面に2本のレーンがあり、`●` が右から左へ流れてくる。`█` ゾーンに到達した瞬間に対応するキーを押せ。

- **J レーン（メイン）** — メロディー。パリィ成功ごとに音程が上がる
- **F レーン（サブ）** — 8ビートの一定リズム。バスドラムの音

#### パリィ判定

- **PERFECT** — ジャストタイミング。残火 +4、コンボ +1
- **Good** — やや早め（遅めは判定が厳しい）。残火 +1、コンボ +1
- **MISS** — パリィゾーンを通過。ダメージを受け、残火 -2、コンボリセット

MISSした弾は灰色になってそのまま流れていく。

### 攻撃

`Space` でボスにダメージ（ATK 3）。攻撃後 0.3秒の硬直 + 0.8秒のクールダウン。

SPACEレーンにクールダウンゲージが表示される。

### 残火バースト

残火ゲージがMAXになると、SPACEレーンが**黄色**に光り「HOLD SPACE!」と表示される。

`Space` を**長押し**すると残火を消費しながら**継続ダメージ**（8 DPS）を与える。離すかゲージが空になると終了。

### 撲殺（残火の業火）

1. パリィを成功させて **残火ゲージ** を溜める
2. `Space` で攻撃してボスのHPを **1** まで削る
3. 残火ゲージが **MAX** の状態で `;` キー → **残火の業火** 発動
4. ボスの `●` マークを1つ削る
5. 全ての `●` を削ると **撲殺**（完全撃破）

ボス「堕落した都知事」は `●●●` を持つ。3回撲殺が必要。

### アイテム

| スロット | アイテム | 効果 |
|---------|---------|------|
| 1 | 緋の雫 | HP +30 回復 |
| 2 | 灰霧の壺 | 敵の攻撃を3秒間減速 |
| 3 | 朽ちた投刃 | ボスに8ダメージ |
| 4 | 受け流しの残滓 | パリィ判定を5秒間拡大 |

`A`/`D` で選択、`S` で使用。消耗品。

### 死んだら

> 「 大いなる死に賛歌を！ 」

パーマデス。最初からやり直し。`R` でリスタート。

## サウンド

パリィ成功で音が鳴る。ちゃんとリズムに乗ると気持ちいい。

| イベント | 音 |
|---------|-----|
| J パリィ | キンキン（金属音、音程がメロディーで変化） |
| F パリィ | ドスドス（バスドラム） |
| PERFECT | より響く金属音 |
| MISS | 小さいドス音 |
| 攻撃 | ズバッ（斬撃音） |
| 残火の業火 | ズバーン |
| ゲーム開始 | ブオン |
