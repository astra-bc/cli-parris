## インストール

### 必要なもの

- Python 3.8 以上
- macOS または Linux（curses 対応ターミナル）
- ターミナルの幅 80 文字以上推奨

### GitHub からインストール

```bash
pip install git+https://github.com/seiyamakino/cli-parris.git
```

### ソースからインストール

```bash
git clone https://github.com/seiyamakino/cli-parris.git
cd cli-parris
pip install .
```

### 開発用インストール（ソース編集が即反映される）

```bash
git clone https://github.com/seiyamakino/cli-parris.git
cd cli-parris
pip install -e .
```

## 遊び方

### 起動

```bash
nononono
```

または:

```bash
python -m cli_parris
```

> **注意**: 起動前に **半角英数入力** に切り替えてください。全角入力だとキーが認識されません。

### タイトル画面

`W` キーを押すとゲーム開始。`Q` で終了。

## 操作方法

| キー | アクション |
|------|-----------|
| `W` | パリィ：上段 |
| `A` | パリィ：左 |
| `S` | パリィ：下段 |
| `D` | パリィ：右 |
| `↑` `↓` | ターゲット切替 |
| `Space` | 攻撃（ターゲット中の敵にダメージ） |
| `Enter` | 残火の血霧（残火MAX＋敵HP1で●を削る） |
| `R` | リスタート（ゲームオーバー/クリア時） |
| `Q` | 終了 |

## ゲームシステム

### パリィ

敵ごとに攻撃バーが表示される。`●` が `█` ゾーンに到達した瞬間に、表示されている方向キー（W/A/S/D）を押せ。

- **PERFECT** — ジャストタイミング。ダメージ0、残火+4
- **Good** — やや早い/遅い。ダメージ軽減、残火+1
- **MISS** — 方向ミスまたはタイミングを外す。フルダメージ、残火-2

### ターゲット

`↑` `↓` でターゲットを切り替える。`|` マークがついている敵がターゲット中。パリィ・攻撃はターゲット中の敵にのみ有効。

### 撲殺（残火の血霧）

1. パリィを成功させて **残火ゲージ** を溜める
2. `Space` で攻撃して敵のHPを **1** まで削る
3. 残火ゲージが **MAX** になったら `Enter` で **残火の血霧** 発動
4. 敵の `●` マークを1つ削る
5. 全ての `●` を削ると **撲殺**（敵を完全に倒す）

ボス「堕落した都知事」は `●●●` を持つ。3回残火を溜めて削れ。

### 死んだら

> 「 大いなる死に賛歌を！ 」

パーマデス。最初からやり直し。`R` でリスタート。
