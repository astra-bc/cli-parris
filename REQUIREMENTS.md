# 灰都殺伐行 (cli-parris) - ゲーム要件定義

## 概要

ターミナル上で動作するローグライク×パリィアクションゲーム。
敵の攻撃として表示される文字列に対してタイミングよくキー入力し、攻撃を受け流す（パリィする）ことがコアメカニクス。

## コンセプト

- **ジャンル**: ローグライク × タイミングアクション（パリィ）
- **プラットフォーム**: ターミナル（TUI）
- **コア体験**: 敵の攻撃を「パリィ」で受け流す緊張感と、連続成功で溜まる残火で反撃する爽快感

## コアメカニクス: パリィシステム

### 操作体系

| キー | アクション |
|------|-----------|
| `W` | パリィ：上段 |
| `A` | パリィ：左 |
| `S` | パリィ：下段 |
| `D` | パリィ：右 |
| `↑` | ターゲットを上の敵に切り替え |
| `↓` | ターゲットを下の敵に切り替え |
| `Space` | 攻撃（現在ターゲット中の敵に攻撃） |
| `Enter` | 残火の憤怒（残火MAX時に使用可能、撲殺判定） |
| `1`〜`4` | アイテムスロット1〜4を使用（将来実装） |

### パリィの仕組み
- 敵の攻撃には**方向**がある（上段/下段/左/右）
- 攻撃の方向に対応するWASDキーをタイミングよく押すことでパリィ成功
- 方向を間違えるとMiss扱い

### 複数敵とターゲット管理
- 1フロアに最大4体の敵が同時に出現する
- 複数の敵が同時または連続で攻撃してくる
- **↑↓**でターゲットを上下に切り替える
- 敵は縦に並んでおり、上下の位置関係でターゲットを管理
- ターゲット中の敵は左に`|`で表示
- ターゲット中の敵に対してパリィ・攻撃が有効
- ターゲット外の敵の攻撃はパリィ不可（ダメージを受ける）
- 敵を倒すとプレッシャーが下がる
- フロアが進むほど敵の数・攻撃頻度が増加

### パリィ判定
- **Perfect**: ジャストタイミングで入力 → ダメージ0、残火大量獲得
- **Good**: やや早い/遅い → ダメージ軽減、残火少量獲得
- **Miss**: タイミングを外す → フルダメージ、残火獲得なし

### アイテムスロット（将来実装）
- 最大4つのアイテムを所持可能（キー`1`〜`4`に対応）
- 戦闘中にリアルタイムで使用可能
- すべて**消耗品**（使い捨て）、使ったらなくなる
- 入手方法:
  - フロアクリア報酬（ランダムドロップ）
  - フロア間のショップで購入（凝血のトークン消費）

#### アイテム例
| アイテム | 効果 |
|---------|------|
| 緋の雫 | HPを一定量回復 |
| 灰霧の壺 | 一定時間、敵の攻撃頻度を下げる |
| 朽ちた投刃 | ターゲット中の敵に即ダメージ |
| 冒涜の火壺 | 全敵にダメージ |
| 受け流しの残滓 | 一定時間、パリィ判定の猶予を拡大 |
| 忘却の石塊 | 次の1回の被ダメージを無効化 |

### 武器（MVP）
- **殺意の出刃包丁** - 固定装備、選択不可
- 武器選択・装備システムは将来実装

### 弾きの残火
- パリィ成功で蓄積される
- 一定量溜まると「残火の憤怒」を発動可能（`Enter`キー）
- 残火は戦闘をまたいで持ち越し可能（要検討）

### 撲殺システム（残火の憤怒）
- 敵はHPバーの下に**撲殺ゲージ**（`●`）を持つ
  - 雑魚: `●`（1個）
  - ボス: `●●●`（複数）
- 残火がMAXの状態で`Enter`を押すと「残火の憤怒」が発動
- 残火の憤怒でターゲット中の敵の`●`を1つ削る（`◯`に変わる）
- すべての`●`が`◯`になったら**撲殺**（敵を完全に倒す）
- HPを0にしても`●`が残っている場合はHP全回復して復活
- ボスは何度も残火を溜めて`●`を削る必要がある

## 敵の設定

### 敵一覧（MVP）

| 名前 | AA | ★ | HP | ATK | ● | 速度 | 役割 |
|------|-----|---|----|----|---|------|------|
| 池袋の豚 | (-.-) | ★☆☆ | 8 | 3 | ● | slow(3秒) | 雑魚。遅い、パリィの練習相手 |
| 渋谷の犬 | (^w^) | ★★☆ | 6 | 5 | ● | normal(2秒) | 雑魚。標準速度 |
| (ボス) 堕落した都知事 | {o_o} | ★★★ | 20 | 8 | ●●● | fast(1.2秒) | ボス。速い攻撃 |

### 攻撃スピード

- 全て等速（一定速度で`●`が進む）
- 敵ごとにスピードが異なる（slow/normal/fast）
- バーの長さは全敵共通（タイミングを合わせやすくするため）

## ゲーム進行

### 構造（1ステージ・MVP）
- 探索要素はなし、戦闘に特化
- いきなりボス戦。説明なし、フロムらしく放り込まれる
- **ボス1体(Daraku Tochiji) + 雑魚2体(Ikebukuro no Buta, Shibuya no Inu)**（計3体）

### フロア間イベント
- フロアクリア後、次のフロアに進む前に以下が発生:
  1. **報酬**: 凝血のトークン + ランダムでアイテムor装備ドロップ
  2. **ビルド選択**: ランダム3択から1つ選択
  3. **ショップ**（一定フロアごとに出現）: 凝血のトークンでアイテム・装備を購入

### 凝血のトークン
- 敵を倒すと獲得
- ショップでの購入に使用
- パーマデスでリセット

### パーマデス
- HPが0になったらゲームオーバー、最初からやり直し
- 死亡時に **「大いなる死に賛歌を！」** と表示
- ランの記録（スコア等）は保存

### 刻印（メタ進行・永続強化）
- ラン終了時（死亡・クリア問わず）にランダムで**刻印**を1つ獲得
- クリア時はより希少な刻印が出やすい
- 刻印はパーマデスでリセット**されない**（永続所持）
- 所持している刻印の効果は**全て常時発動**
- 各刻印には**3つの効果**が付いている
- ランを重ねるほど刻印が増え、プレイヤーの基礎力が底上げされる

#### 刻印例
| 刻印 | 効果1 | 効果2 | 効果3 |
|------|-------|-------|-------|
| 灰に沈んだ瞳 | パリィ猶予+微量 | 被ダメ-1 | 残火蓄積+5% |
| 朽ちぬ楔の欠片 | HP上限+10 | フロアクリア時HP微回復 | 凝血のトークン獲得+10% |
| 忌まわしき残響 | 攻撃力+3 | Perfect時に敵の攻撃遅延 | 残火蓄積+10% |
| 夜霧の産声 | 被ダメ-2 | アイテム効果+20% | ショップ割引10% |
| 名も無き者の指輪 | HP上限+5 | 攻撃力+2 | パリィ猶予+微量 |
| 深淵の残り香 | 残火蓄積+15% | 残火満タン時攻撃力UP | 被ダメ+2（デメリット） |

## ローグライク要素: ランダムビルド

### 戦闘間の成長
- フロアクリア時にランダムなビルド選択肢が提示される
- 例: スキル獲得、ステータス強化、パッシブ効果
- 毎回異なる組み合わせで戦略が変わる

### ビルド例（MVP後に拡張）
- パリィ判定の猶予時間を広げる
- 弾きの残火の蓄積速度UP
- 反撃ダメージ増加
- HP回復効果

### 表示要素
- 現在のフロア数
- 敵の一覧（縦並び。各敵: AA + 名前 + HPバー + 撲殺ゲージ`●` + 攻撃力`★`）
- 攻撃の方向インジケーター（up/down/left/right）
- 敵ごとの攻撃タイミングバー（独立した行。`●`が`[░░█░░]`ゾーンに入ったらパリィ可能。`█`でPerfect、`░`でGood）
- ターゲット中の敵は左に`|`で表示
- ターゲット切替はカーソル上下
- プレイヤーHP・残火（1行にまとめて表示）
- 操作キー表示

## 技術要件

- **言語**: 未定（Python curses / Rust crossterm 等を検討）
- **依存**: 最小限、標準ライブラリ中心
- **対応OS**: macOS（開発環境）、Linux 互換を意識

## MVP スコープ

### 含むもの
- [ ] パリィ戦闘（タイミング入力 → 判定 → ダメージ計算）
- [ ] 弾きの残火蓄積 → 強攻撃発動
- [ ] 1ステージ構成（ボス+雑魚2体、いきなりボス戦）
- [ ] パーマデス（死んだら最初から）
- [ ] フロアクリア時のランダムビルド選択（3種から1つ）
- [ ] 複数敵の同時出現（最大4体、複数レーン攻撃）
- [ ] 基本的な敵バリエーション（3種 + ボス1種）
- [ ] 撲殺システム（残火の憤怒で敵の●を削る）
- [ ] 刻印システム（ラン終了時にランダム獲得、永続強化、効果3つ付き）
